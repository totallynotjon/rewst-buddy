- **Live Jinja preview** — preview how the active linked template renders against a real workflow
  execution's context, updating as you type, via "Preview Jinja Render" (`Ctrl+Shift+Enter` /
  `Cmd+Shift+Enter`). Opens as native editor panes with a "Pick Jinja Preview Context" command for
  layering manual variable overrides on top of the execution's context.
