- **Large AI tool results** - Rewst Buddy now keeps oversized AI tool output in an in-memory cache and returns a preview plus a result id, so Cage-Free Rewsty can page through or search the full result with the new `buddy_result_read` tool instead of rerunning the request. Nothing is written to disk; the cache size is configurable via `rewst-buddy.ai.toolResultCacheLimitMB` (default 500 MB).
