- **Cage-Free Rewsty now confirms every Rewst mutation each time.** GraphQL
  mutations and workflow edits, auto-layouts, and runs are no longer remembered
  per resource for the session — each one shows its inline confirmation
  individually, so running a workflow (or any change) always asks. To stop being
  asked for a tool, use VS Code's own _Allow in this Session_ option on the
  confirmation, which allow-lists the tool itself.
