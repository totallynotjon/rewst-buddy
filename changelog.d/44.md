- **Cage-Free Rewsty now asks for approval before every Rewst mutation and
  workflow run, every time.** GraphQL mutations and workflow edits, auto-layouts,
  and runs each pop an **Allow / Cancel** notification before they run. The
  prompt is the extension's own UI, shown from inside the tool call rather than
  through VS Code's tool-confirmation affordance, so it can't be auto-approved or
  allow-listed away — even when another agent (e.g. Copilot agent mode) drives
  the tool. Running a workflow or changing one never happens unattended.
