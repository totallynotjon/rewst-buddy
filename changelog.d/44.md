- **Cage-Free Rewsty now confirms every Rewst mutation and workflow run, every
  time.** GraphQL mutations and workflow edits, auto-layouts, and runs each show
  an inline confirmation before they run — the extension no longer remembers
  approvals per session, and it ships these tools as **not eligible for
  auto-approval** (`chat.tools.eligibleForAutoApproval`), so the prompt can't be
  allow-listed away (even in another agent like Copilot agent mode). Running a
  workflow or changing one never happens unattended unless you opt back into
  auto-approval or VS Code's Autopilot / Bypass-Approvals mode.
