category: Changed

- **Running a Rewst workflow from chat now asks every time.** Cage-Free Rewsty's
  `buddy_workflow_run` confirmation is no longer remembered per workflow — because
  a run executes real automation, each run is confirmed individually. Approving an
  edit or auto-layout on a workflow no longer silently lets it be run, and a run
  approval no longer grants edit access. Editing and auto-layout are still
  remembered per workflow for the session.
