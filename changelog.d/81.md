- **Workflow action edits no longer corrupt task input or drop integration overrides** — editing a task through `buddy_workflow_edit` now preserves the task's per-task integration override (pack override) instead of silently reverting it to the org default, and a task `input` supplied as a JSON-encoded string is parsed back into an object instead of being stored as a character-indexed blob. The workflow graph view also surfaces a task's integration override so an edit visibly keeps it.
