- **Cage-Free Rewsty keeps its todo list current and finishes it** — the AI steering now tells the assistant to mark each todo in progress when it starts and completed the moment it finishes (instead of batching status updates to the end), to drive the list all the way to completion, and to reconcile against the recorded todo state — not its own memory — before declaring a task done, so it no longer reports work as finished while items are still unchecked or incomplete.
